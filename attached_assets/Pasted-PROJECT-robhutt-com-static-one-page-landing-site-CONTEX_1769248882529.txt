PROJECT: robhutt.com (static one-page landing site)
CONTEXT:
Build a very simple, high-impact, minimal one-page landing site for Rob Hutt. Dark theme. Cinematic. Clean. No clutter. No blog, no CMS, no auth, no database. Just a static landing page with four “launch windows” (cards) that will later link out to: Rob Hutt Films, Hutt Studio, CharacterX, IceMaker, FlashBuzz. For now use image placeholders only.

PRIMARY GOAL:
A polished, Apple-like minimal homepage that feels premium and confident. The copy below is the source of truth. Use UK English spelling.

STACK REQUIREMENTS:
- Vite + React + TypeScript
- Tailwind CSS
- shadcn/ui (optional but preferred for Button/Card components)
- No backend required
- Must deploy as a static site (build output only)

DESIGN DIRECTION:
- Dark theme by default
- Minimal, bold typography
- Lots of negative space
- Subtle gradient background and faint film grain texture effect (CSS only, no images needed)
- Subtle hover states on cards and buttons
- Responsive: excellent on mobile, tablet, desktop
- Max content width: ~1100px
- Use a single accent colour for highlights (subtle, not neon)
- Avoid long dashes (—). Use normal hyphens or rewrite.

INFORMATION ARCHITECTURE (single page):
1) Top nav (very minimal)
   - Left: “Rob Hutt”
   - Right: “Work” anchor, “Contact” anchor
   - Sticky with subtle blur

2) Hero section
   - Title: Rob Hutt
   - Subhead lines:
     “Award-winning filmmaker.”
     “Founder of Hutt Studio.”
   - Short intro paragraph:
     “I work at the intersection of story, strategy, and systems, helping people and organisations communicate with clarity, credibility, and intent.”
   - Soul Heart Hands line (large and spaced):
     “Soul. Heart. Hands.”
   - Primary CTA button: “View the work” (scrolls to Work)
   - Secondary CTA button: “Get in touch” (scrolls to Contact)

3) Soul Heart Hands section (3 columns on desktop, stacked on mobile)
   - Soul heading + paragraph
   - Heart heading + paragraph
   - Hands heading + paragraph

4) Work section (the launch windows)
   - A grid of cards (2 columns desktop, 1 column mobile)
   - Each card includes:
     - Image placeholder (simple div, 16:9, subtle border, label like “Image placeholder”)
     - Title
     - One or two short paragraphs
     - A button: “Explore” (for now href “#” or a placeholder route)
   - Cards:
     A) Rob Hutt Films
     B) Hutt Studio
     C) CharacterX
     D) IceMaker
     E) FlashBuzz
   - If the grid looks better with 4 items, keep 4 in the main grid and add FlashBuzz as a full-width card beneath. Prioritise visual balance.

5) Availability section

   - CTA: “Start a conversation” button (scrolls to Contact)

6) Contact section
   - Minimal contact block, no form for now
   - Show an email placeholder line like: “Email: hello@robhutt.com” (hard-coded placeholder)
   - Add small note: “If the work feels aligned, I’m always open to a conversation.”

7) Footer
   - Minimal copyright line: “© {current year} Rob Hutt”
   - Small links: “Work”, “Contact”
